# Expense Tracker App

A minimal, offline-capable expense tracking web application.

## Features
- Add, edit, delete expenses
- View spending trends
- Filter by category/month
- Export to CSV
- Works offline
- Dark/light theme

## How to Use
Open `index.html` in your browser.
